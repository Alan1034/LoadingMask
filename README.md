# LoadingMask

一个带有动态Loading的遮罩层 <br/>
A Vue loading mask

![image-20211116112215815](https://raw.githubusercontent.com/Alan1034/PicturesServer/main/PicGo_imgs/202111161127946.png)

用法：

     <LoadingMask 
     :text="percentage"  // [String, Number]
     :maskVisable="maskVisable" // Boolean
     />
